## Варіант завдання
Розрахунок параметрів виконується на основі номера залікової книжки (№ = 5122$):

1.  **C5** (Операція): $5122 : 5 => остача 2$
    * Дія: $C = A + B$.
2.  **C7** (Тип даних): $5122 : 7 => остача 5$
    * Тип елементів матриці: `char`.
3.  **C11** (Дія з матрицею): $5122 : 11 => остача 7$
    * Завдання: Обчислити суму найбільших
      елементів в стовпцях матриці з
      непарними номерами та найменших
      елементів в стовпцях
      матриці з парними номерами.

## Опис реалізації

Програма виконує наступні кроки:
1. Ініціалізує та заповнює випадковими значеннями дві цілочисельні матриці $A$ та $B$.
2. Обчислює матрицю $C$ шляхом додавання двох  матриць $A$ та $B$ типу `char` використовуючи їх символьні коди.
3. Проходить по кожному стовпчику матриці $C$ для розрахунку:\
   – найбільших елементів у стовпцях з непарними номерами;\
   – найменших елементів у стовпцях з парними номерами.
4. Виводить результат у консоль.
